## Installation

To Install & Run This Project You Have To Follow Thoose Following Steps:

```sh
git clone https://github.com/rhishi-kesh/Tailwind-Practice.git
```

```sh
composer install
```

```sh
php artisan key:generate
```

```sh
php artisan serve
```
